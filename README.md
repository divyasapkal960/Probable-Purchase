# Probable-Purchase
A Deep Dive into Customer Purchase Predictions

Probable Purchase:
-	Objective: A deep dive into customer purchase predictions to build a machine learning model that predicts whether customers will make their next purchase within a specified period (in this case, 90 days from their last purchase).
-	Technologies Used: Python, Machine Learning Algorithms (Logistic Regression, Decision Tree, Random Forest, K-Means)
